# GitHub User Finder

**Автор:** Тарлина Снежана Васильевна

## 📋 Описание

GitHub User Finder - это десктопное GUI-приложение для поиска пользователей GitHub с возможностью сохранения избранных пользователей. Приложение предоставляет удобный интерфейс для просмотра детальной информации о пользователях GitHub и управления списком избранных.

### Основные возможности:
- 🔍 Поиск пользователей GitHub по имени
- 👤 Отображение подробной информации о пользователе
- 🖼️ Просмотр аватара пользователя
- ⭐ Добавление пользователей в избранное
- 💾 Сохранение избранных пользователей в JSON
- 🌐 Быстрый переход к профилю на GitHub
